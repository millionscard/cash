## Getting started

### Install dependencies

```bash
npm install
```

### Start development environment

```bash
npm run ios
```

```bash
npm run android
```

### Start development environment of the docs

```bash
npm run docs
```

### Deploy docs

```bash
npm run docs:deploy
```
